Cuadro comparativo entre las sentencias para consultas en MySQL vs Mongo DB

Sentencias en MYSQL Sentencias en MONGODB

1. Listado de todos los usuarios con solo los nombres, apellidos y edad, que tengan 20 años de edad. Con MySQL:

SELECT nombres, apellidos, edad FROM users WHERE edad = 20;

1. Listado de todos los usuarios con solo los nombres, apellidos y edad, que tengan 20 años de edad. Con MongoDB:

db.users.find( {edad:20}, { nombres: 1, apellidos: 1, edad: 1, _id: 0});

2. Listado de todas las mujeres en la Base de Datos que tengan entre 20 y 30 años de edad. Con MySQL:

SELECT nombres, genero, edad FROM users WHERE genero = 'M' AND edad >= 20 AND edad <=30;

SELECT * FROM users WHERE genero = 'M' AND >= 20 AND EDAD <=30;

2. Listado de todas las mujeres en la Base de Datos que tengan entre 20 y 30 años de edad. Con MongoDB:

db.users.find({genero: 'M', edad: {$gte: 20, $lte: 30}}, { nombres: 1, apellidos: 1, genero: 1, edad: 1, _id: 0});

3. ¿Quién es la persona con menos edad de la Base de Datos? Con MySQL:

SELECT nombres, MIN(edad) FROM users;

SELECT * FROM users ORDER BY edad ASC LIMIT 1;

3. ¿Quién es la persona con menos edad de la Base de Datos? Con MongoDB:

db.users.find().sort({ edad: 1 }).limit(1);

4. ¿Cuántos usuarios hay registrados en la Base de Datos?. Con MySQL:

SELECT COUNT(id) AS Total FROM users;

4. ¿Cuántos usuarios hay registrados en la Base de Datos?. Con MongoDB:

db.users.count();

5. Traer los 5 primeros usuarios de la base de datos. Con MySQL:

SELECT DISTINCT * FROM users ORDER BY id ASC LIMIT 5;

SELECT TOP 5 * FROM users ORDER BY id;

5. Traer los 5 primeros usuarios de la base de datos . Con MongoDB:

db.users.find().limit(5);

6. Traer los 10 últimos usuarios de la base de datos. Con MySQL:

SELECT DISTINCT * FROM users ORDER BY id DESC LIMIT 10;

6. Traer los 10 últimos usuarios de la base de datos. Con MongoDB:

db.users.find().sort({_id: -1}).limit(10);

7. Listar usuarios que su correo finalice en .net. Con MySQL:

SELECT * FROM users WHERE correo LIKE '%.net';

7. Listar usuarios que su correo finalice en .net. Con MongoDB:

db.users.find({ correo: { $regex: /\.net$/i } });

8. Listar usuarios que NO vivan en Colombia. Con MySQL:

SELECT * FROM users where pais <> 'colombia';

8. Listar usuarios que NO vivan en Colombia. Con MongoDB:

db.users.find({ pais: { $ne: "colombia" } });

9. Listar usuarios que NO vivan en Ecuador y Panamá. Con MySQL:

SELECT * FROM users where pais <> 'ecuador' AND pais <> 'panama';

9. Listar usuarios que NO vivan en Ecuador y Panamá. Con MongoDB:

db.users.find({ pais: { $nin: ["ecuador", "panama"] } });

10. ¿Cuántos (en números) usuarios son de Colombia y les gusta el rock?. Con MySQL:

SELECT COUNT(id) AS Total FROM users WHERE pais = 'colombia' AND musica = "rock";

10. ¿Cuántos (en números) usuarios son de Colombia y les gusta el rock?. Con MongoDB:

db.users.count({ pais: "colombia", musica: "rock" });

11. Actualizar el género musical de todos los usuarios de la Base de Datos de "metal" a "carranga". Con MySQL:

UPDATE users SET musica = 'carranga' WHERE musica = 'metal';

11. Actualizar el género musical de todos los usuarios de la Base de Datos de "metal" a "carranga". Con MongoDB:

db.users.updateMany( { musica: "metal" }, { $set: { musica: "carranga" } } );

12. Listado de hombres que les guste la "carranga" sean de Colombia, y tengan entre 10 y 20 años de edad. Con MySQL:

SELECT * FROM users WHERE genero = 'H' AND musica = 'carranga' AND edad >= 10 AND edad <= 20;

12. Listado de hombres que les guste la "carranga" sean de Colombia, y tengan entre 10 y 20 años de edad. Con MongoDB:

db.users.find({ genero: "H", musica: "carranga", pais: "colombia", edad: { $gte: 10, $lte: 20 } });

13. Actualizar todos los usuarios que tengan 99 años, su nuevo género musical favorito será la "carranga". Con MySQL:

UPDATE users SET musica = 'NULL' WHERE musica = 'carranga' AND edad = 99;

13. Actualizar todos los usuarios que tengan 99 años, su nuevo género musical favorito será la "carranga". Con MongoDB:

db.users.updateMany( { edad: 99 }, { $set: { musica: "carranga" } } );

14. Listar todos los usuarios que su nombre inicie con "a","A". Con MySQL:

SELECT * FROM users WHERE nombres LIKE 'a%' OR nombres LIKE 'A%';

14. Listar todos los usuarios que su nombre inicie con "a","A". Con MongoDB:

db.users.find({ nombres: { $regex: /^a|^A/ } });

15. Listar todos los usuarios que su apellido finalice en "z", "Z". Con MySQL:

SELECT * FROM users WHERE apellidos LIKE '%z' OR apellidos LIKE '%Z';

15. Listar todos los usuarios que su apellido finalice en "z", "Z". Con MongoDB:

db.users.find({ apellidos: { $regex: /z$|Z$/ } });

16. Actualizar los usuarios que tengan 50 años de edad, su nuevo género musical será NULL. Con MySQL:

UPDATE users SET musica = 'NULL' WHERE edad = 50;

16. Actualizar los usuarios que tengan 50 años de edad, su nuevo género musical será NULL. Con MongoDB:

db.users.updateMany( { edad: 50 }, { $unset: { musica: "" } } );

db.users.updateMany( { edad: 50 }, { $set: { musica: "NULL" } } );

17. Listar todos los usuarios que su género musical sea igual a NULL. Con MySQL:

SELECT * FROM users WHERE musica = "NULL";

17. Listar todos los usuarios que su género musical sea igual a NULL. Con MongoDB:

db.users.find({ musica: { $exists: false } });

db.users.find({ musica: "NULL"});

18. ¿Cuál es el resultado de la suma de todas las edades de la Base de Datos?. Con MySQL:

SELECT SUM(edad) FROM users;

18. ¿Cuál es el resultado de la suma de todas las edades de la Base de Datos?. Con MongoDB:

db.users.aggregate([ { $group: { _id: null, total: { $sum: "$edad" } } } ]);

19. ¿Cuántos usuarios tenemos registrados de "ecuador"?. Con MySQL:

SELECT COUNT(id) AS Total FROM users WHERE pais = 'ecuador';

19. ¿Cuántos usuarios tenemos registrados de "ecuador"?. Con MongoDB:

db.users.count({ pais: "ecuador" });

20. ¿Cuántos usuarios son de "colombia" y les gusta el vallenato?. Con MySQL:

SELECT COUNT(id) AS Total FROM users WHERE pais = 'colombia' AND musica = "vallenato";

20. ¿Cuántos usuarios son de "colombia" y les gusta el vallenato?. Con MongoDB:

db.users.count({ pais: "colombia", musica: "vallenato" });
